### ğŸ”¹ GET `/api/date`

`/api/2015-12-25`

**Response:**


`/api/1451001600000`

**Response:**

```json
{
  "unix": 1451001600000,
```

`/api/invalid-date`

**Response:**

```json
{
  "error": "Invalid Date"
}
```
